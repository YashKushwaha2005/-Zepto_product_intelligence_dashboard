# Zepto Quick Commerce — Product Intelligence Dashboard

## Problem Statement
Quick commerce managers had no unified platform to monitor 
product portfolio health, pricing strategy, and inventory 
availability across 14 categories.

## Objective
Build a 3-page Power BI dashboard connected to SQL data 
source providing product intelligence, pricing analysis, 
and inventory monitoring in one place.

## Tools Used
- SQL — Data extraction and transformation
- Power BI — Multi-page dashboard and visualization
- Miro — Workflow diagrams

## Dashboard Pages
- Page 1 — Product Intelligence (14 categories, 3,732 SKUs)
- Page 2 — Pricing and Discount Analysis
- Page 3 — Inventory and Stock Analysis

## Key Metrics
- Total SKUs — 3,732
- Average Discount — 7.62%
- Out of Stock Rate — 12.14%
- Total Available Units — 15K

## Key Findings
- Cooking Essentials and Munchies have highest OOS at 64 each
- Meats Fish and Eggs has highest avg discount at 11.03%
- 49.36% of SKUs carry less than 5% discount
